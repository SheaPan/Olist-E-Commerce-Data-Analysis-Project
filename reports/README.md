# Reports Folder

This folder contains all final deliverables, including Power BI dashboards, presentations, and written reports.

## Subfolders

### `/powerbi`
- The final Power BI `.pbix` file(s).
- Contains all dashboards, visuals, and data models.

### `/written_report`
- Written documentation detailing methodology, analysis, and key findings.
- Includes explanations of customer, product, sales, and delivery insights.

## Notes
- These files represent the **final outputs** of the project.
- Use them to present, review, or validate the analysis.
